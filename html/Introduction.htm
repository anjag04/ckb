ARTICLE 1

html contains two types of tags

1) paired tags
2)unpaired tags

HTML has 2 rules to write tags:
- Normal Tags (most of the tags): <TAGNAME>content</TAGNAME>.
- Self Closing Tags: <TAGNAME attribute = "value" />. Don't get confused, look at that: <img src="imgPath/imgName.imgExtension"/>
the "img" is the TAGNAME.
the "src" is the image source file.
the "imgPath/imgName.imgExtension" is the value of the "src" attribute.
We use the "img" tag to display an image in our website.

w3c validator to check your code for errors, miss used tags or unsupported tags. this will also tell you if a tag is deprecated and no longer used, or isn't compatible with all browsers.

ARTICLE 3

web hosting is where your website will reside so that other people can get to it. Typically, people will buy the services of a company that specializes in that area, but you could also host your website yourself if you have the hardware and technical knowledge (or if you can Google ).

nobody "pays a domain name" per se. One can enlist the services of a DNS provider so that other people can find their website more easily. Domain Name Service (DNS) providers assign names to IP addresses that computers use to find each other. Consider how easy it is to remember and type www.google.com as opposed to remembering 64.233.173.193 . That's thanks to a DNS that translates our words into magic internet numbers .
